About ChurnPredict AI

A machine learning powered web application to predict and prevent customer churn

Project Objective

This system uses machine learning to predict whether a customer will churn (stop using a service) before it actually happens — allowing businesses to take proactive retention actions.

Three ML models are trained and compared: Logistic Regression, Random Forest, and XGBoost — with XGBoost achieving the best performance.

Problem Statement

Companies lose significant revenue every year due to customer churn. Traditional methods are reactive — businesses only realize a customer has left after they're already gone.

This system solves that by predicting churn before it happens so businesses can act in time.

System Modules
Module 1: Data Management

Handles dataset upload, preprocessing, cleaning, and storage in SQLite.

Module 2: Model Training

Trains and compares Logistic Regression, Random Forest, and XGBoost models.

Module 3: Prediction Engine

Predicts churn probability and assigns Low / Medium / High risk badges.

Module 4: Feature Importance

Shows top features driving each prediction using bar charts.

Module 5: Retention System

Suggests discounts, actions, and sends automated retention emails.

Module 6: Analytics Dashboard

Displays churn trends, statistics, and exportable reports.
